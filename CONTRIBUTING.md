# Contributing

## Setup
```bash
npm install
make dev
npm test
```

## Branch Naming
- `feature/description`
- `bugfix/description`
- `hotfix/description`

## Commit Format
```
type(scope): description
```

## Code Style
- Follow TypeScript/Angular/NestJS best practices
- Write tests for new features
- Use meaningful names
- Keep code clean and simple
